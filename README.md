STAC STOCKTAKE
**************

Local deployment
-----------------

1. Install the requirements

    .. code-block::

        pip install -r requirements.txt

2. Copy and update `.stac_stocktake.yml.tmp`

    .. code-block::

        cp .stac_stocktake.yml.tmp .stac_stocktake.yml

3. Run the `stac_stocktake`_

    .. code-block::

        python -m stac_stocktake
